Опис програми
Ця програма реалізує класичний метод комп'ютерної стеганографії LSB (Least Significant Bit) мовою Python. Вона дозволяє приховувати довільні текстові повідомлення всередині графічних файлів формату PNG без видимих змін для людського ока.

Основні можливості:

Приховування даних: Текст конвертується у бінарний код і записується у молодші біти каналів RGB.

Витягування даних: Програма зчитує біти з пікселів та реконструює початковий текст.

Автоматизація: Скрипт автоматично визначає шляхи до файлів, що усуває помилки "File not found".

Чистий код: Логіка роботи з бітами реалізована "з нуля" без використання готових стеганографічних бібліотек (лише Pillow для відкриття зображень).

Для роботи програми необхідний встановлений інтерпретатор Python 3 та бібліотека Pillow.

Функціональні компоненти
hide_message(cover_path, stego_path, message_text)

Відкриває оригінальне зображення (cover_path).

Додає до тексту спеціальний бінарний маркер (END_MARKER), щоб позначити кінець повідомлення.

Модифікує молодші біти пікселів і зберігає результат у файл stego_path (формат PNG).

Виводить статистику: скільки біт використано та який відсоток ємності зображення зайнято.

extract_message(stego_path)

Сканує пікселі обробленого зображення.

Збирає молодші біти та формує з них байти.

Шукає стоп-маркер і декодує прихований текст у формат UTF-8.

_to_bits та _from_bits

Допоміжні функції для низькорівневої роботи з бітовими операціями (зсув бітів, бітові маски).

Інструкція із запуску
Попередня підготовка:

Переконайтеся, що встановлено бібліотеку: pip install Pillow

У папці з програмою (lab02) має знаходитися файл-зображення з назвою test.png.

Метод 1: Через вбудований термінал VS Code
Це надійний спосіб запуску, який дозволяє бачити всі системні повідомлення.

У Visual Studio Code відкрийте термінал (Ctrl + `).

Перейдіть у папку лабораторної роботи (якщо ви ще не там):

Bash

cd lab02
Запустіть скрипт командою:

Bash

python lab_2_kozhan.py
Програма автоматично виконає демонстрацію: сховає вбудований у код текст та одразу спробує його розшифрувати.

Метод 2: Через кнопку "Запуск" (Run)
Найпростіший спосіб для швидкої перевірки.

Переконайтеся, що файл lab_2_kozhan.py відкритий і активний на екрані.

Натисніть на трикутник Run (Play) у правому верхньому куті вікна VS Code.

Результат роботи (звіт про приховування та розшифрований текст) з'явиться у нижній панелі TERMINAL.

Примітка: Щоб змінити секретне повідомлення, відкрийте код файлу lab_2_kozhan.py, знайдіть рядок secret_message = "..." у кінці файлу, змініть текст на власний і запустіть програму знову.
